# 方式一

通过终端指令去启动

首先在终端切换到你的MongoDB的安装目录的bin目录下

1. 首先在终端切换到你的MongoDB的安装目录的bin目录下
2. mongod --dbpath C:\MongoDB\Server\4.4\data
1
2
3

注意：这种方式窗口千万不要关闭，否则无法连接，如果对数据库的操作结束，可以打开该控制台通过 Ctrl + C 关闭。

# 方式三

  1. 先检查自己的配置信息，如若一切无误后，则使用管理员模式开启cmd，不要直接打开cmd。

  2. 然后在cmd中转跳至自己MongoDB的bin目录，使用我的举例是：

    cd` `"C:\Program Files\MongoDB\Server\3.4\bin"
    1

  3. 然后cmd写入服务

    mongod --dbpath "C:\MongoDB\Server\4.4\data" --logpath "C:\MongoDB\Server\4.4\log\mongodb.log" --install --serviceName "MongoDB"
    1
    mongod.exe --logpath C:\MongoDB\Server\4.4\log\mongodb.log  --logappend --dbpath C:\mongodata --directoryperdb  --serviceName MongoDB --install

mongod --dbpath "C:\mongodata" --logpath "C:\MongoDB\Server\4.4\log\mongodb.log" --install --auth --serviceName "MongoDB"

mongod -f C:\MongoDB\Server\4.4\bin\mongo.conf
mongod --config C:\MongoDB\Server\4.4\bin\mongo.conf --install
mongod.exe --dbpath c:\mongodata

//删除服务
mongod --logpath "C:\MongoDB\Server\4.4\log\mongodb.log" --logappend --dbpath "C:\MongoDB\Server\4.4\data" --directoryperdb --serviceName "MongoDB" --serviceDisplayName "MongoDB" --remove
//重新安装  注意：去除--directoryperdb 命令
mongod --logpath "C:\MongoDB\Server\4.4\log\mongodb.log" --logappend --dbpath "C:\MongoDB\Server\4.4\data" --serviceName "MongoDB" --serviceDisplayName "MongoDB" --install
//添加权限启动
mongod --logpath "C:\MongoDB\Server\4.4\log\mongodb.log" --logappend --dbpath "C:\MongoDB\Server\4.4\data" --serviceName "MongoDB" --auth --serviceDisplayName "MongoDB" --install

  5. 开启完可进行简单的测试是否开启
