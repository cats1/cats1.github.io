# canvas

  • Canvas是 HTML5 新增的一个标签，它的主要作用是画矢量图。使用Javascript 脚本来绘制图像。canvas能做游戏，能做特别炫的效果，主要是为画图而生。
  • Canvas是一个实时渲染的图片，可以实现不刷新页面，对画布进行实时渲染（runtime)。

使用标签 <canvas> 在页面创建一个 canvas 画布，一个 2D 平面，并渲染出指定的画面，如形状、图像、文字、图案等。

判断用户浏览器是否支持 < canvas >标签，支持 < canvas> 的浏览器会只渲染 < canvas> 标签，而忽略其中的替代内容。不支持 < canvas> 的浏览器则会直接渲染替代内容。

<!--显示文本-->
<canvas>你的浏览器不支持 canvas，请升级你的浏览器。</canvas>
<!--显示图片-->
<canvas><img src="./img.jpg" alt=""> </canvas>

# 创建 canvas

使用标签 <canvas> 创建画布，该元素（DOM 元素）具有以下属性

  • width 画布的宽度
  • height 画布的高度

可以通过 CSS 或 JavaScript 设置画布大小

const canvas = document.querySelector('canvas');
const ctx = canvas.getContext('2d');
let windowWidth = document.documentElement.clientWidth;
let windowHeight = document.documentElement.clientHeight;
// 设置画布大小（满屏）
canvas.width = windowWidth;
canvas.height = windowHeight;

canvas 使用坐标系定位图像，在二维图像中，左上角为 (0,0) 坐标值依次向右侧（x 值）和向下方（y 值）增大。

该 DOM 元素具有方法 getContext 获取 canvas 的内容，返回一个对象，具有更丰富的方法，可以将该对象理解为一支「画笔」，调用它的不同方法并传递不同的参数可以画出不同的形状。

# 保存与恢复画布状态

使用 ctx 的方法 save() 和 restore() 来保存和恢复画布状态，更方便地在不同样式间切换，以绘画出不同的图形。

# 擦除画布 clearRect

使用 ctx 的方法 clearRect(posX, posY, width, height) 擦除画布的特定区域。

// 将参数设置为画布的尺寸，擦除整个画布
ctx.clearRect(0, 0, c.width, c.height);

# 基本图形

# 矩形

使用 ctx 方法 fillRect() 或 strokeRect() 绘制实心矩形或空心矩形，传递不同的参数以定制矩形外观 fillRect(posX, posY, width, height)。

canvast 提供了三种方法绘制矩形：

  • fillRect(x, y, width, height)：绘制一个填充的矩形。
  • strokeRect(x, y, width, height)：绘制一个矩形的边框。
  • clearRect(x, y, widh, height)：清除指定的矩形区域，然后这块区域会变的完全透明。

默认颜色是黑色，可以通过以下属性进行修改：

  • 属性 fillStyle 设置填充颜色
  • 属性strokeStyle 设置边框颜色属性

# 路径

  • 可以通过 ctx 多种方法创建路径 Path (opens new window) 以绘制出一个复杂的形状。
  • 使用方法 beginPath() 声明需要创建一个新的路径
  • 使用方法 moveTo(x, y) 和 lineTo(x, y) 创建路径，前一个方法确定路径开始点，相当于将「画笔」移动到指定的坐标；后一个方法基于当前坐标和该方法给出的坐标之间创建一条路径
  • 方法 fill() 或 stroke() 结束路径绘制，并创建实心或空心的图形
// 绘制路径
ctx.beginPath(); // 声明开始绘制线段
ctx.moveTo(75, 75); // 设定起始点，相当于移动（提起的）画笔
ctx.lineTo(125, 75); // 画线
ctx.lineTo(125, 125); // 画线
ctx.lineTo(75, 75); // 画线
ctx.fill(); // 根据路径绘制出填充图形

# 弧线或圆形

使用 ctx 方法 arc 绘制弧形，使用弧度单位

context.arc(x,y,r,sAngle,eAngle,counterclockwise);

参数 描述
x 圆的中心的 x 坐标。
y 圆的中心的 y 坐标。
r 圆的半径。
sAngle 起始角，以弧度计。（弧的圆形的三点钟位置是 0 度）。
eAngle 结束角，以弧度计。
counterclockwise 可选。规定应该逆时针还是顺时针绘图。False = 顺时针，true = 逆时针。

  • 中心：arc(100,75,50,0Math.PI,1.5Math.PI)
  • 起始角：arc(100,75,50,0,1.5*Math.PI)
  • 结束角：arc(100,75,50,0Math.PI,1.5Math.PI)

arc(x,y,r,start,stop)

ctx.beginPath();   // 绘制路径前需要声明，避免与前一个路径相连
// 画一个圆，半径大小是 30
ctx.arc(300, 500, 30, 0, Math.PI * 2, false);
ctx.strokeStyle = 'green'
ctx.stroke();

# 颜色

使用 ctx 属性 fillStyle 或 strokeStyle 设置填充或线框颜色，支持使用 CSS 规范声明的约 140 种颜色名称 (opens new window)或 RGB 十六进制码。

# 文本

  • 使用 ctx 的方法 fillText(string, x, y) 或 strokeText(string, x, y) 绘制空心或实心的文本 (opens new window)，其定位坐标基于字符串左下角的位置。

  • 使用 ctx 的属性 font 设置字体样式与字体大小，默认使用 CSS font 属性相同的语法，默认值字体和大小是 10px sans-serif

  • 使用 ctx 的属性 textAlign 设置文本对齐方式，可选的值包括

    • start 默认值

    • end

    • left

    • right

    • center

  • 使用 ctx 的属性 fillStyle 或 strokeStyle 设置填充颜色

  • 使用 ctx 的属性 lineWidth 设置边框宽度

# 绘制图像 drawImage

使用 ctx 的方法 darwImage() (opens new window) 在画布上绘制图片、其他画布或视频，以实现复制、增添特效等功能。也可以增加或减少图像的尺寸以绘制图像的某些部分。

// 剪切图像，并在画布上定位被剪切的部分：
ctx.drawImage(img, sx, sy, swidth, sheight, x, y, width, height);

# 图像数据 ImageData

画布可以作为像素图片使用，使用 ctx 方法 getImageData (opens new window) 从画布获取一个图像数据对象，也可以使用 ctx 方法 createImageData (opens new window) 初始化一个空的图像数据对象；修改后的数据使用 ctx 方法 putImageData (opens new window) 设置回去。

图像数据对象包含图像三种属性

  • 宽度 width
  • 高度 height
  • 数据 data 存储每个像素的 RGB 数据和 Alpha 值（范围是 0-255，其中 0 是透明的，255 是完全可见的），一般使用 Uint8ClampedArray 数组表示，数组每四个元素表示一个像素点信息，即 RGBA 值

用以下方法获取或修改图片数据：

  • createImageData（width， height, imageData) 初始化一个空的图像数据对象
  • getImageData(x, y, width, height) 复制画布上指定矩形的像素数据，起始点为 (x,y) ，宽为 width 高为 height
  • putImageData(imagedata, dx, dy, dirtyX, dirtyY, dirtyWidth, dirtyHeight) 存储数据到背景画布
let imgData = ctx.getImage(0, 0, c.width, c.height);

function paintGreen(imagData) {
  let numPix = imgData.data.length / 4;
  // 将画布中每隔 10 个像素设置为绿色
  for (let i = 0; i < numPix; i++) {
    if(i % 10 === 0) {
      // 将第 i 个像素的 Green 值设置为 255（需要背景原色为白色，否则还需要编辑 red 值和 blue 值
      imgData.data[i*4 + 1] = 255;
      // 将第 i 个像素的 Alpha 值设置为 255（非透明）
      imgData.data[i*4 + 3] = 255;
    }
  }
  // 将修改后的图像数据存储回背景画布
  ctx.putImageData(imgData, 0, 0);
}

paintGreen(imgData);
// 将彩色图片转换为黑白（灰）图片
let img = new Image();
img.src = "test.jpg";
img.onload = function() {
    ctx.drawImage(img, 0, 0, c.width, c.height);
    imgData = ctx.getImageData(0, 0, c.width, c.height);
    gray(imgData); // 调用灰度函数
}

// 灰度函数
function gray(imagData) {
    let numPix = imgData.data.length / 4;

    for (let i = 0; i < numPix; i++) {
        let gray = (imgData.data[i*4] + imgData.data[i*4+1] + imgData.data[i*4+2])/3
        imgData.data[i*4] = gray;
        imgData.data[i*4+1] = gray;
        imgData.data[i*4+2] = gray;
        imgData.data[i*4 + 3] = 255;
    }
    ctx.putImageData(imgData, 0, 0);
}

# 变换操作

# 缩放 scale

使用 ctx 的方法 scale(x,y) 将该命令后再创建的形状的 x 和 y 值乘以给定系数，即放大相应的倍数

# 移动 translate

使用 ctx 的方法 translate(x,y) 将该命令后再创建的形状沿水平方向移动 x 个像素，沿垂直方向移动 y 个像素。

# 旋转 rotate

使用 ctx 的方法 rotate(angleRadians) 将对象（通常）绕中心转动特定弧度

💡 度数与弧度的关系 radian = degree * (Math.PI/180)

# 动画

使用方法 requestAnimationFrame ，其作用相当于一个定时器（类似于 setTimeout 和 setInterval） 让浏览器以一帧为单位循环执行画面（渲染）操作。

方法 requestAnimationFrame 接受一个动画执行函数作为参数，这个函数在每一帧都会被执行一次，可以对每一帧画面进行渲染，并根据条件判断是否结束，如果动画没有结束，则继续调用 requestAnimationFrame 并将函数自身作为参数传入（循环调用）。

function animationWidth() {
  let div = document.getElementById('box');
  div.style.width = parseInt(div.style.width) + 1 + 'px';

  if(parseInt(div.style.width) < 200) {
    requestAnimationFrame(animationWidth)
  }
}

animationWidth();

# 处理鼠标输入

画布元素 <canvas> 也可以接收鼠标相关事件，关键是要确定用户在画布上的哪个位置点击。

鼠标点击事件 event 对象的属性 clientX 和 clientY 记录了点击相对于浏览器窗口全局中的位置，再使用每个元素的属性offsetLeft和offsetTop 得到画布元素相对于浏览器左上角 (0,0) 的偏移量，从 clientX 和 clientY 中减去 offsetLeft 和 offsetTop 值，就可以得到用户点击相对于画布的位置。

function handleMouseClick(event) {
        x = event.clientX - c.offsetLeft;
        y = event.clientY - c.offsetTop;
        console.log("x, y:" + x + "," + y);
}

c.addEventListener("click", handleMouseClick, false);
