# vscode 插件

# 代码片段类插件

# 1、JavaScript(ES6) snippets

这个插件为 JavaScript、TypeScript、HTML、React 和 Vue 提供了 ES6 的语法支持

# 2、HTML Boilerplate

# 自动补全类插件

# 1、Path Intellisense

智能路径提示、自动路劲补全

# 2、Npm Intellisense

用于在import语句中自动填充npm模块

# 3、Vetur

Vue 的语法高亮、智能感知、Emmet 等；

# 4、css-auto-prefix

自动添加 CSS 私有前缀。

# 功能增强插件

# 1、Debugger for Chrome

让 vscode 映射 chrome 的 debug 功能，静态页面都可以用 vscode 来打断点调试，需要另外配置，有点麻烦；

# 2、Project Manager

在多个项目之前快速切换的工具 工程项目过多时，shift+cmd+p(shift+ctrl+p) 然后输入project，第一次选择edit Project编辑自己的工程项目，之后就可以直接选择open打开你的项目

# 3、Change Case

转换命名风格。

# 4、Sass

# Git相关插件

# 1、GitLens

详细的 Git 提交日志。

Git 重度使用者必备，尤其是多人协作时：哪一行代码，何时、何人提交都有记录。

# 2、Git blame

# 3、Git History

查看版本号，文件对比

# 代码检测相关插件

# 1、ESlint

ESlint 接管原生 js 提示，可以自定制提示规则；

# 2、Regex Previewer

实时预览正则表达式的效果。

# 3、stylelint

CSS / SCSS / Less 语法检查

# 4、code spell checker

对基本关键字拼写校验

检查你的英文单词拼写是否有误，如果有库里不存在的单词则会下下波浪线

# 美化编辑器类

# 1、Bracket Pair Colorizer

给嵌套的各种括号加上不同的颜色。

# 2、vscode-icons

改变编辑器里面的文件图标

# 3、Guides

指导线，当前所在的级别缩进线会变红，当前在哪一级一目了然。

# 快捷操作插件

# 1、View In Browser

在浏览器里预览网页必备。

# 2、Highlight Matching Tag

高亮对应的 HTML 标签 以及 标识出对应的各种括号。

# 3、Auto Rename Tag

自动修改匹配的 HTML 标签。

# 4、Auto Close Tag

自动闭合标签

# 5、Turbo Console Log

快捷添加 console.log，一键 注释 / 启用 / 删除 所有 console.log。

ctrl + alt + l 选中变量之后，使用这个快捷键生成 console.log

alt + shift + c 注释所有 console.log

alt + shift + u 启用所有 console.log

alt + shift + d 删除所有 console.log

# 6、Live Server

这个插件基本功能是预览网页，但它的特点是：会将网页在本地服务器上预览，最重要的是代码保存之后，浏览器自动刷新

# 7、Image Preview

鼠标悬浮在链接或者装订线(gutter)左边可以预览到图片

# 8、CSS Peek

定位 CSS 类名。

# 格式化操作插件

# 1、Prettier

格式化插件

# 注释类

# 1、Document This

快速注释

# 2、fileheader

顶部注释模板，可定义作者、时间等信息，并会自动更新最后修改时间 ctrl + alt + i你可以在头部插入注释，ctrl + s保存文件后，自动更新时间和作者。

# 3、Better Comments

使注释有人性化的高亮显示

# 提示类

# 1、Import Cost

在编辑器中显示引入包的大小

# 2、filesize

在底部状态栏显示当前文件大小，点击后还可以看到详细创建、修改时间

# 其他插件

# 1、carbon-now-sh

将代码分享为图片（图片的格式可以为 png 和 svg）

# 2、Codelf

Codelf 是一个用来给变量命名的网站，你只要输入你想起的中文名，它就会给你提供很多建议的命名：

# 3、settings sync

在不同电脑间同步你的插件。

Visual Studio代码设置同步，需要github账号辅助的
